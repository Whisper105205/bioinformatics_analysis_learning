## 代码
```R
# 开启保存设备
png("p1.png",width=800,height=600,res=300)
# 绘图
library(ggplot2)
p1<-ggplot(data,aes(x=,y=)+
geom_point(color="",alpha=,size=)+
...+
theme_bw())
print(p1)#必要，否则为空
# 关闭
dev.off()
```

```R
ggsave(file="文件名.格式"，plot=,width=,height=)
```
